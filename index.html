Add Your Training Data

Customize your model with domain-specific knowledge

1

Prepare Your Training Data

Create a JSONL file with your training examples in the OROBOROS format:

{"text": "Your training content here..."} {"text": "More training content..."} {"text": "Each line is a separate example"}
2

Create a Modelfile

Build upon the base model with your custom system prompt:

FROM oroboroslabs/claude-opus:latest SYSTEM """ Your custom system prompt here. Add domain-specific instructions. Include your training context. """ PARAMETER temperature 0.7 PARAMETER num_ctx 8192
3

Build Your Custom Model

Use Ollama to create your personalized model:

ollama create my-custom-claude -f Modelfile ollama run my-custom-claude
4

Upload Memory Module

Connect to the OROBOROS memory system for persistent knowledge:

# Download memory module ollama pull oroboroslabs/memory-module:latest # Link to your model ollama create my-claude-with-memory -f Modelfile-memory
